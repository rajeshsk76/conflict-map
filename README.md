# EntropyMap

Live site: https://entropymap.live

This repository hosts a conflict and infrastructure alert dashboard built from RSS feeds and published with GitHub Pages.

## Files
- `index.html` — dashboard page
- `conflict_map.html` — interactive map
- `update_conflict_map.py` — generator script
- `requirements.txt` — Python dependencies
- `.github/workflows/update-map.yml` — scheduled auto-update workflow

## Notes
This is an automated news-headline visualization prototype, not verified intelligence.
